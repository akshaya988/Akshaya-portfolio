⚙️ Setup & Run Locally
# Install dependencies
npm install

# Run development server
npm run dev

👉 Open: http://localhost:5173

📦 Build for Production
npm run build
npm run preview
🚀 Deployment

This project is deployed using Vercel.

To deploy manually:

npm i -g vercel
vercel
